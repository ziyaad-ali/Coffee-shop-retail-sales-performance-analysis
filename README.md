# Coffee-shop-retail-sales-performance-analysis
Analyzed pre-processed coffee shop retail sales data sourced from kaggle using Excel, SQL Server, and Power BI to deliver clear insights through interactive dashboards.
This project examines coffee shop retail sales to understand revenue patterns, product-wise performance, and demand trends.
A pre-processed dataset was analyzed using Excel and SQL Server to generate insights. 
Power BI dashboards were created to present key metrics, time-based trends, and category performance in a clear, decision-focused manner.
